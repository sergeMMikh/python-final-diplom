Дипломная работа к профессии Python-разработчик

«API Сервис заказа товаров для розничных сетей».

Список предлагаемых опций

- Ссылка на документацию: /doc.

Для пользователей:

- Регистрация пользователя с последующим подтверждением электронного адреса /api/v1/user/registe.

- Аутентификация пользователя /api/v1/user/login.

- Восстановление пароля с использованием электронного адреса пользователя /api/v1/user/password_reset.

- Создание и редактирование контактов /api/v1/user/contact.

- Редактирование данных пользователя /api/v1/user/details.


Для покупателей:

- Получение списка магазинов /api/v1/shops.

- Поиск товаров по идентификатору магазина и категории /api/v1/products?shop_id&category_id.

- Создание корзины с покупками и редактирование её содержимого /api/v1/basket.

- Получение списка заказов и возможность разместить заказ /api/v1/order.


Для магазинов:

- Обновление прайса /api/v1/partner/update.

- Полчение и редактирование статуса магазина /api/v1/partner/state.

- Полчение магазином списка заказов /api/v1/partner/orders.


Для суперпользователей

- Административная панель /admin/.
